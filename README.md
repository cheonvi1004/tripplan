# Tripplan - 여행 코스 메이커

나만의 여행 코스를 예쁜 이미지로 만들어보세요.

## 주요 기능

- 드래그로 경유지 순서 변경
- 6가지 컬러 테마 선택
- 카카오맵 연동
- PNG 이미지 다운로드
- Firebase 방문자 카운터

## 개발

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build
```

## 배포

GitHub Pages로 자동 배포:

```bash
npm run deploy
```

## Firebase 설정

`public/firebase-config.js` 파일의 `YOUR_API_KEY`, `YOUR_SENDER_ID`, `YOUR_APP_ID`를 실제 Firebase Console 값으로 교체하세요.
